# Triple Espresso

## Nome do Projeto

Cafeteria Triple Espresso

## Descrição

Página web responsiva da cafeteria Triple Espresso, desenvolvida com HTML5 e CSS. O projeto contempla a incorporação de vídeos via iframes, formulário de reserva de mesa, aplicação da metodologia BEM para nomenclatura de classes e organização de arquivos, com versionamento e hospedagem em repositório no GitHub.

## Tecnologias e técnicas utilizadas

- HTML5 semântico
- CSS (Flexbox, posicionamento absoluto, pseudo classes)
- Iframes (incorporação de vídeos do YouTube)
- Formulários HTML (inputs de texto, número, data/hora, e-mail e checkbox)
- Metodologia BEM (mixin BEM e estrutura de arquivos BEM Flat)
- Google Fonts (Inter e Noto Serif)
- GitHub

## Funcionalidades

- Navegação por âncoras entre as seções da página
- Seção de receitas com vídeos incorporados
- Formulário de reserva de mesa com validação nativa do navegador

## Plano de melhorias

- Implementação de ações dinâmicas com JavaScript
- Integração com banco de dados para armazenamento e gerenciamento das informações enviadas pelo formulário de reserva
